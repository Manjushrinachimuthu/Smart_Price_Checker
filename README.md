# Smart Price Checker

Smart Price Checker is a full-stack web application that helps users compare product prices across different stores.

## 🚀 Features
- Compare product prices from multiple stores
- Track price history
- User login and authentication
- Shopping cart functionality
- Price alerts and notifications

## 🛠 Technologies Used

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Spring Boot
- Java
- REST API
- Maven

### Database
- JSON / File storage

## 📂 Project Structure

```
Smart_Price_Checker
│
├── pricefrontend   # React frontend
├── pricebackend    # Spring Boot backend
└── README.md
```

## ▶️ How to Run

### Frontend

```
cd pricefrontend
npm install
npm start
```

### Backend

```
cd pricebackend
mvn spring-boot:run
```
